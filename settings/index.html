Diagnostic report

Generates a snapshot of one Omnik inverter: settings, cached state, capability values and the result of probing both the binary TCP protocol on port 8899 and the HTTP /js/status.js fallback. The report stays on your device; nothing is auto-sent. Use it when reporting an issue.

Use this when the inverter is paired with Homey, even if it's currently marked unavailable.

Use this when pairing fails so option A isn't available yet. HTTP /js/status.js is probed with no auth and with the admin/admin default; the binary protocol on TCP/8899 can only be probed once the WiFi-stick S/N is known.

Paste the copied report into a new GitHub issue. The report contains your inverter's local IP, MAC address and serial number — these are only useful on your home network, not a security risk to share. The HTTP password is automatically redacted. If you'd rather strip the other identifiers before sharing, edit the text after pasting.
